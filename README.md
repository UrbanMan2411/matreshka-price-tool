# MATRЁSHKA — генератор прайс-листа из Excel

Веб-сервис: загружаешь `.xlsx` в формате прайса **MATRЁSHKA** (ТМ ООО «КубаньБытХим») → получаешь готовый брендированный **PDF**. Всё считается прямо в браузере — файл никуда не отправляется.

🟥 **Прод:** https://matreshka-price-tool.vercel.app/

Это «брат-близнец» сервиса Green Panda ([greenpanda-price-tool](https://github.com/UrbanMan2411/greenpanda-price-tool)) — та же кодовая база, перекрашенная под красно-золотую фолк-палитру Матрёшки.

---

## Что умеет
- Парсит xlsx целиком в браузере (SheetJS) — категории, товары, цены.
- **Достаёт фото товаров прямо из xlsx** (картинки вшиты в файл; читаем `xl/media/*` + якоря из `xl/drawings/drawing1.xml` через JSZip).
- Генерит многостраничный PDF (pdf-lib) — A4 landscape, по строке на товар, шапка с реквизитами, заголовки категорий, фото-колонка.
- Блок выбора фона-водяного знака: **Стандартный / Свой (загрузка) / Без фона** + ползунок прозрачности.
- Кириллица — через вшитый шрифт Manrope (статические Regular/Bold).

## Стек
React 18 · Vite 5 · pdf-lib + @pdf-lib/fontkit · SheetJS (xlsx) · JSZip. Без бэкенда — чистая статика, деплой на Vercel.

## Структура
```
public/brand/
  Manrope-Regular.ttf, Manrope-Bold.ttf   # шрифты для PDF (кириллица)
  bg.jpg                                   # дефолтный фон-водяной знак
src/
  App.jsx              # UI: загрузка, выбор фона, кнопка генерации
  index.css            # стили + брендовая палитра (CSS-переменные)
  lib/
    parseXlsx.js       # парсинг xlsx + извлечение вшитых фото
    buildPdf.js        # вёрстка PDF (pdf-lib): шапка, таблица, фото, футер
```

## Формат входного xlsx (важно!)
Парсер **сам находит строку-шапку** по слову `наименование`, поэтому работают оба формата (у Матрёшки шапка в строке 8, у Панды — в 4-й). Колонки (1-based):

| Кол. | Смысл |
|------|-------|
| 1 | Фото упаковки (вшитая картинка) / название категории |
| 2 | наименование |
| 3 | краткое описание |
| 4 | объём |
| 5 | артикул |
| 6 | штрих-код |
| 7 | паллет, шт |
| 8 | кол-во в коробе |
| 9 | цена |

**Строка-категория** — где в кол. 1 есть текст, а кол. 2 и 5 пустые (напр. «Крем-мыло»). Тестовый файл: `ПРАЙС ДЛЯ ПЕРЕВОДА В PDF MATRESHKA …xlsx`.

## Где менять бренд
- **Цвета** — CSS-переменные `--green-900/700/500` и т.д. в `src/index.css` (имена остались от базы Панды, значения = бордо/красный/золото).
- **Палитра PDF** — константы `GREEN9/GREEN7/GREEN5/PAPER…` вверху `src/lib/buildPdf.js`.
- **Шапка/футер PDF, заголовок колонки цены** — функции `drawHeader`, `drawTHead`, `drawFooter` в `buildPdf.js`.
- **Текст-вордмарк** «MATRЁSHKA» — в `drawHeader` (PDF) и `<span className="wordmark">` в `App.jsx` (UI).

## Заметки
- Дефолтный фон `bg.jpg` сейчас — нейтральный мягкий пейзаж (на ~8% почти незаметен). Под Матрёшку можно положить свой `public/brand/bg.jpg` или выбрать «Без фона» в интерфейсе.
- PDF получается на 8 страниц при 68 SKU — нормально.
